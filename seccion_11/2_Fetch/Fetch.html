Uso de fetch() y archivos JSON

¿Qué es fetch()?

Es una función moderna de JavaScript que permite hacer peticiones HTTP. Devuelve una Promesa, lo que facilita trabajar con datos de forma asíncrona (sin bloquear la página).

¿Qué es una Promesa?

Una Promesa en JavaScript representa un valor que puede estar disponible ahora, en el futuro o nunca. Es una forma de manejar operaciones asincrónicas. Una promesa tiene tres estados posibles:

  • Pendiente: Aún no se ha completado.
  • Resuelta (cumplida): Se completó exitosamente.
  • Rechazada: Ocurrió un error.

Con .then() manejamos la resolución y con .catch() el rechazo.

Pasos clave:

  1. Se usa fetch('persona.json') para solicitar el archivo.
  2. .then(res => res.json()): transforma la respuesta a un objeto JSON.
  3. .then(data => ...): accede al contenido y lo usa en el HTML.

Ejemplo de código:

fetch('persona.json')
  .then(respuesta => respuesta.json())
  .then(datos => {
    document.getElementById("nombre").textContent = datos.nombre;
  })
  .catch(error => {
    console.error("Error al cargar el archivo JSON:", error);
  });

Explicación:

  • fetch(): Inicia la petición HTTP.
  • .then(res => res.json()): Convierte la respuesta a objeto JSON.
  • .then(datos => ...): Usa los datos convertidos.
  • .catch(): Captura errores si el archivo no se encuentra o hay un fallo en la red.

Fragmento de JSON esperado:

{
  "nombre": "Pedro Toledo",
  "edad": 35,
  "esEstudiante": false
}
